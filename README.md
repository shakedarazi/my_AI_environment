# my_AI_environment

Personal AI toolkit — skills, agents, commands, hooks, rules, and configurations.

## Directory Tree

```
my_AI_environment/
├── agents/                  # Top-level agent definitions
│   └── subagents/           # Specialized subagents
├── skills/                  # Claude Code skills (.md)
├── commands/                # Slash commands
├── hooks/                   # Event hooks (session, tool, etc.)
├── rules/                   # Behavioral rules and guidelines
├── claude-mds/              # CLAUDE.md files for different projects/contexts
├── constitutions/           # Constitution MDs (values, principles)
├── speckit/                 # Spec templates and project scaffolding
└── ai/                      # AI-related utilities and experiments
```
